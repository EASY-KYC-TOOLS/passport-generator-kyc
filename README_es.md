## ✨ ¿Qué es EasyKYC?

**EasyKYC** es una plataforma de **generación de documentos KYC de nivel financiero**. Proporcionamos perfiles de prueba 100% sintéticos y seguros para ingenieros, investigadores de seguridad y equipos de cumplimiento.
