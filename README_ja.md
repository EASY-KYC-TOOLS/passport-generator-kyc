## ✨ EasyKYC とは?

**EasyKYC** は、金融グレードの **KYC ドキュメント生成プラットフォーム** です。エンジニア、セキュリティ研究者、コンプライアンスチーム向けに、プライバシーが保護された 100% 合成テストプロファイルを提供します。

---

## 🚀 主な機能

### 🛂 **パスポートジェネレーター**
15 か国以上で **有効な ICAO 9303 MRZ コード** 付きの標準準拠のパスポートを生成:
- 🇺🇸 米国（2007 / 2021）
- 🇬🇧 英国
- 🇫🇷 フランス
- 🇲🇽 メキシコ
- 🇯🇵 日本
- 🇸🇬 シンガポール
- 🇮🇳 インド
- 🇨🇦 カナダ
- 🇧🇾 ベラルーシ
- 🇺🇦 ウクライナ
- 🇺🇿 ウズベキスタン
- 🇹🇼 台湾

### 🪪 **ID カード & 運転免許証ジェネレーター**
適切なセキュリティ機能を備えた国民 ID カードと運転免許証を作成:
- 🇯🇵 日本国民 ID カード
- 🇺🇸 米国運転免許証（MS / WA / NY）
- 🇯🇵 日本運転免許証
- 標準準拠のチップおよび生体データ

### 💰 **銀行明細書ジェネレーター**
住所証明の金融書類を生成:
- Chase 銀行（米国）
- Bank of America（米国）
- Citibank（英国）
- SoCalGas（米国）
- TEPC（日本）
- 千葉電力（日本）
- CityGas（シンガポール）

## 🔒 技術標準

| 標準 | 実装 |
|:---|:---|
| **MRZ フォーマット** | ICAO 9303 準拠（TD3 パスポート、TD1 ID） |
| **チェックディジット** | ICAO 仕様に従って検証 |
| **EXIF メタデータ** | 認証された携帯電話カメラのメタデータ注入 |
| **ドキュメント構造** | 国際標準（ISO/IEC 7501） |
| **データプライバシー** | 100% 合成 — 実際の PII は関与しない |
| **規制遵守** | GDPR / CCPA / APPI 準拠 |
